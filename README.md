# irc in go

IRC! Internet relay chat! It's how hackers communicate when they don't 
wanna be overheard.

## Usage of ./bunnyirc:

  -n string

    	Nickname (default "chameleon")

  -p int

    	Port to use (default 6667)

  -s string

    	Server to connect to (default "chat.freenode.net")

  -u string

    	Username (default "chameleon")

  -z	Use TLS
